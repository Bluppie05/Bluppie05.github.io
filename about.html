About Me

Hi, I am Simon L. Better known as Bluppie05, I am a back-end/fullstack web developer from the Netherlands and this is my story of how it all started.

How I started

I started programming when I was about 7 years old. My parents gave me a book about Scratch and Python. I read it and immediately became hooked, I started creating small games in a visual programming tool for kids called Scratch. After I got bored with Scratch, I started looking at the Python section of the book, I created really small games using Turtle and had alot of fun doing it. When I was about 9, my dad learned me some HTML, I created terrible looking but still fun websites using HTML-only(no css). When I was about 10, I dropped programming, I had alot going on and was hooked to the all new game called Minecraft. In 2018, I decided to pick up programming again and learned more than ever, I already knew the basics of HTML so that is what I continued in, I started learning Javascript and CSS and a while later, I learned PHP. With a good friend, I set up a name which we published our software under called BitBiz. After having used PHP for a long time, I moved on to Python-Flask to bring my programs closer to the Operating System itself and learned even more about computers and programming in general. After I learned Flask, I realized that I always enjoyed the Javascript part of web development the most, that is when I decided to learn NodeJS to be able to write most of my programs for the most part in Javascript, I fell in love with this language and I'm still largely using it today.

My Skills and Experiences

Over the years, I have largely worked with Javascript, however, next to Javascript, I know: Python3, PHP, Go, C, C#, Java, VisualBasic, PowerShell and some more. In almost every language I have mostly used web-frameworks and Discord tools, I have fallen in love with web-develpment and discord-bot development because you can create alot with relatively little code. The thing about Web-development I enjoy the most is the Back-end part, where your code is critical for the front-end to survive, serving the processed data over the internet.
